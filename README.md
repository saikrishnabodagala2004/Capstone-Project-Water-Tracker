Capstone Project – Water Footprint Tracker

The Water Footprint Tracker helps users understand the water impact of everyday products including foods, drinks, and clothing. It’s built to raise awareness about sustainable freshwater usage by presenting simple, easy-to-use data on how much water goes into items we use every day.

Tech Stack

React.js for the frontend

Firebase Firestore for database

Firebase Auth for authentication

Firebase Hosting for deployment

React Router

Firestore SDK

GitHub for version control

Getting Started

This project was bootstrapped with Create React App.
In the project directory, you can run:

npm start

Runs the app in development mode.
Open http://localhost:3000
 in your browser.
The page reloads on save and shows lint errors in the console.

npm test

Launches the test runner in watch mode.

npm run build

Builds the app for production in the build folder.
React is optimized and bundled for best performance.

npm run eject

This copies configuration files so you can customize the setup.
Use only if necessary—this action is irreversible.
